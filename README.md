wget -N --no-check-certificate https://github.com/admnn/ServerStatus-Toyo/raw/master/status.sh && chmod +x status.sh
```

下载脚本后，根据需要安装客户端或者服务端：

bash status.sh c
 
# 显示服务端管理菜单
bash status.sh s
```
